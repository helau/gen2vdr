Ausgabe von evtest - 60 mögliche TastenCodes für den X10 Receiver
Zuordnung TastenCodes zu FB-Tasten bei Modul Option keychange=1
Für Digitainer X10 FB mit 44 Tasten plus Scrollwalze, die Auf/Ab ersetzten soll.
Die Scrollwalze liefert jedoch undefinerte codes für Auf/Ab die abhängig von den
vorhergehenden Bewegungen sind (u.a. Typ2,code0(X) mit versch. values für Abwärts 
und code 272 LeftBtn bzw. code 275 SideBTN für Aufwärts)
Daher für Auf/Ab besser P+/- (Chan+/-) verwenden!!
